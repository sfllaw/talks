# Talks

## Go

Go talks are formatted according to
[golang.org/x/tools/cmd/present](https://pkg.go.dev/golang.org/x/tools/cmd/present).

Use [golang.org/x/tools/cmd/present](https://pkg.go.dev/golang.org/x/tools/cmd/present)
to present them locally:

``` console
sfllaw@framework:~/Projects/talks$ go tool present -notes
2026/05/13 15:17:46 Open your web browser and visit http://127.0.0.1:3999
```

Or link to them over the web at
[http://talks.godoc.org/github.com/sfllaw/talks/_PATH_](http://talks.godoc.org/github.com/sfllaw/talks/PATH).
